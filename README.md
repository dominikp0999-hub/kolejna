# Minimalistyczne portfolio fotografa

Prosty, elegancki i responsywny starter strony fotografa (HTML/CSS/JS).
Zawiera:
- Hero z dużym zdjęciem (Unsplash)
- Siatkę portfolio z lightboxem
- Sekcję o mnie i formularz kontaktowy (Formspree)

Jak uruchomić:
1. Rozpakuj i otwórz `index.html` lokalnie, lub
2. Wgraj wszystkie pliki na GitHub i opublikuj przez GitHub Pages (instrukcja poniżej).

Uwaga: W produkcji podmieniaj linki obrazów na własne CDN/Cloudinary i zmień `action` formularza na swój endpoint (Formspree lub własny backend).
